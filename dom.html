Document Object Model Manipulation

Document Object Model

The Document Object Model, or DOM, is a tree hierarchy representation of an HTML document where each branch contains a node that represents the document, an element, or content as an object that makes it possible to be manipulated programmatically with a language like JavaScript. Nodes can be selected, created, appended, removed, have their content manipulated, etc. using the DOM APIs various interfaces.

Some of the more common DOM manipulation methods used include:

Selecting an element by an id in the HTML
document.getElementById("elementID");

Creating an new element with a specified tag
document.createElement("tagName");

Appending a node to a parent node
parentName.appendChild("childName");

Allows for reading or modifying the HTML markup contained in the element
element.innerHTML

Adding an event listener (onclick, onload, onfocus, etc.) to an element
element.addEventListener("eventType", handler);

Using these methods, various aspects of the document can be dynamically modified in-place, making the DOM API very useful for having a page respond to user interaction or changes in data.
