JSON

JavaScript Object Notation, or JSON, allows for JavaScript objects to be represented as a string to store and exchange object data. JSON syntax resembles JavaScript's object literal syntax, combining data in key/value pairs and separating each data element by a comma. Arrays are denoted by brackets while objects are contained within braces, and there can be multiple levels of nested arrays, objects, and properties. JSON strings may be "pretty-printed" to be more human-readable, but they are generated as a single-line string by default. JSON is not necessarily limited to JavaScript; many languages can interpret and process strings in the JSON format.

[
  {
    color: "red",
    value: "#f00"
  },
  {
    color: "green"
    value: "#0f0"
  },
  {
    color: "blue",
    value: "#00f"
  }
]

Parse

JSON.parse() is a method that takes a JSON string and parses it (deserializes) to construct a JavaScript object that is contained in the object literal as described by the JSON syntax. Using JSON.parse(), it is possible to return JavaScript that was serialized to its original object form so that data can be passed between server and client in text and then reused as a data structure.

let myObject = JSON.parse(jsonString[, optional reviver]);

Stringify

JSON.stringify() makes it possible to convert an object (serialize) to the JSON syntax so that it can be passed between client and server, especially via APIs. It can be called with a replacer function or array of values to replace any values that may be considered extraneous, and the string or number of characters used for whitespace can be defined to make it more readable for humans.

var APIresponse = JSON.stringify(serverDataObject[,replacer function/array[, space string/number of whitespaces]]);
