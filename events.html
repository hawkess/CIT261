Events

The Events interface allows for JavaScript to trigger actions in response to events that take place in a document. There are a large number of events pertaining to various aspects of a document, including mouse events, focus events, animation events, form events, etc. A full list of events can be found here. Coupled with an eventListener, these events form the framework to tie JavaScript functions to the elements of the page and make it interactive and respond to changes in state.
