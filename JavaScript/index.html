JavaScript - Variables, Loops, Conditional Statements, Functions, Arrays, & Associative Arrays

Variables

Variables are used to give a name to data so that it can be used again later. In Javascript, variables are created with the var keyword and are mutable so the data can be changed. Since Javascript is dynamically and weakly typed, there is no requirement to declare a type. The type of the variable will change based on the data assigned to it.

var testVariable = 1;
testVariable = "This is now a string.";

Conditional Statements

Conditional statements provide for a method to control the execution of statements based on the result of a boolean expression, explicit or implicit. Using switch, if, then, and else values directly, or comparison operators such as >, <,>=, <=,==,===, !=, !==. When the conditional statement evaluates to true, the code in the block is executed, otherwise it is skipped over. else allows for code to be executed in the case where the condition is not met. Switch statements provide a way of making a selection based on a list of conditions. When one condition evaluates to true, the code in that block is executed until a break statement is encountered.

if (condition) {
  code to execute
} (optional) else {
  code to execute in the negative case;
}

If can be chained with else to form else if statements with an added condition.

if (condition) {
  code to execute
} else if (secondary condition){
  code to execute in the negative case;
}

Loops

Loops can be used to perform a set of instructions repeatedly until a condition is met. Javascript supports for, while, and do...while loops. for loops make it easy to repeat an action for a discrete number of iterations and while/do...while loops allow for indefinite continuation until a condition is met, such as a sentinel value being set. Conditional statements are an integral part of the loop control structure.

for (initial expression; condition; post expression){ code to execute; }
for (var i = 0; i < 5; i++) { console.log(i); } //prints 0-4 to the console, at which point i is no longer < 5 and execution stops.

while (condition) { code to execute; };
while (i !=5) { i++; } //continues until i==5

Functions

Functions are a way of naming blocks of code to be executed. These blocks of code are declared, given a name, and any variables that are needed are identified as parameters, then the block of code can be reused by calling the name given to them and passing any parameters needed in parentheses. This allows for a common action to be repeated multiple times without replicating the code.

Arrays & Associative Arrays

Arrays are a data structure that allow multiple values to be stored in the same container. They can be used to store a list of strings, numbers, Objects, etc. Array notation can be used to access each element using brackets, starting from index 0 for the first element.

var arrayExample = ["this", "is", "an", "example"];
arrayExample[1] returns the value "is".

Associative arrays are arrays that use strings as keys to access elements instead of numerical indice. Javascript does not support associative arrays, but Objects and the dot notation can be used to simulate associate arrays.

var notValid = ["firstname" = "John", "lastname" = "Smith"];
