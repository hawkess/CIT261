HTML5 Media Tags - Canvas, Audio, Video

HTML5 Media Tags

HTML5 offers native support for embedding media and drawing graphics in browsers using the audio, video, and canvas tags. These elements are part of the HTML5 standard and should work across all modern browsers, but the tags provide an easy fallback method that uses the body of the tag to define a message for unsupported browsers.

Canvas

Canvas creates an area that can be used to draw shapes, paths, and text with JavaScript, allowing for all sorts of creative usage within a webpage.
